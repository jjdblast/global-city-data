Cities of the world

A data visualisation by admataz for pictographic.io

The idea here is to simplify the view a diverse set of data and allow comparison of subjects (cities) via single facets of their data. I made this visualisation as a tool-sharpening exercise - read more on my thoughts in producing this and see the full source code on the pictographic github page. The data was downloaded from London Datastore. For a more complete explanation of the figures, sources and units of measurement refer to the original New York City Global City database.
